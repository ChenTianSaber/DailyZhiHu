## 首先

日报知乎是一个基于Android平台的第三方知乎日报，界面UI参考自ios版的知乎日报。  

知乎日报API来自项目[ZhihuDailyPurify](https://github.com/izzyleung/ZhihuDailyPurify)的[wiki](https://github.com/izzyleung/ZhihuDailyPurify/wiki/%E7%9F%A5%E4%B9%8E%E6%97%A5%E6%8A%A5-API-%E5%88%86%E6%9E%90).    

感谢开源项目：[Glide](https://github.com/bumptech/glide)  
感谢开源项目：[retrofit](https://github.com/square/retrofit)  
感谢开源项目：[SlidingMenu](https://github.com/jfeinstein10/SlidingMenu)
